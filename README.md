Bekostigde hogescholen met (toekomstige) Ad-opleidingen kunnen deelnemen. Het platform stimuleert naamsbekendheid, kennisdeling en samenwerking tussen onderwijs, werkveld en overheid, en ondersteunt de Landelijke Ad-overleggen (LAdO‚Äôs) bij de gezamenlijke profilering van Ad-opleidingen.

### Vraag van de opdrachtgever
De bestaande website is niet echt gebruiksvriendelijk en kan een herontwerp gebruiken. Maak een vernieuwde, moderne website met een leuke huisstijl en goede navigatie voor de gebruiker.

### Ontwerpkeuzes
Het ontwerp dat ik heb gekozen is in de afgelopen sprints deels aangepast, maar de basis is hetzelfde gebleven. Ik heb mijn ontwerpkeuzes gewijzigd nadat uit een toegankelijkheidstest bleek dat het kleurcontrast onvoldoende was. Daarom heb ik de knoppen een andere kleur gegeven om het contrast te verbeteren. Goed contrast is belangrijk, omdat ook gebruikers met een visuele beperking de knoppen duidelijk moeten kunnen zien en gebruiken.

### Toegankelijke HTML
De HTML is zo opgebouwd dat de website toegankelijk is voor alle gebruikers, ongeacht eventuele beperkingen.

### Mobile-first
De website is volledig mobile-first ontwikkeld en werkt responsive, zodat de website op elk scherm te gebruiken is.

### JavaScript sorteer functie
De website heeft een sorteer functie op de nieuws pagina, dit is gemaakt met JavaScript. Met deze functie kunnen gebruikers de nieuwsartikelen op meest relevant en meest recent sorteren.
